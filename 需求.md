# 功能描述
该项目是基于 iFlow 官方 Python SDK 开发的 OpenAI 兼容式 OpenAPI 接口适配层，核心作用是将 iFlow 内置 AI 模型包装为严格遵循 OpenAI 规范接口，让所有支持 OpenAI OpenAPI 标准的工具、客户端、代码库无需任何改造，即可直接调用 iFlow 的 AI 模型能力。脚本仅做请求 / 响应格式的纯转换与接口桥接，不修改、不阉割 iFlow 模型的任何原生能力，是连接 iFlow 生态与 OpenAI 生态的轻量中间层。

# 使用说明
使用示例：
1、接口地址：POST http://127.0.0.1:11666/v1/chat，调用端配置的 Base URL 为http://127.0.0.1:11666/v1；
2、鉴权规则：请求头必须携带Authorization: Bearer 111222333444555666，脚本仅校验该值是否完全匹配，无实际的鉴权、权限控制逻辑；
3、模型规则：请求体model字段必须为iflow，脚本仅校验字段值格式，不做实际的模型路由，始终调用 iFlow 内置的 AI 模型；
4、请求限制：仅支持stream=true的流式请求，不支持非流式调用，与 iFlow SDK 的流式响应特性保持一致；
5、响应格式：返回结果严格遵循 OpenAI chat.completion.chunk规范，包含id、object、created、model、choices等核心字段，错误信息也按 OpenAI 标准格式封装。
